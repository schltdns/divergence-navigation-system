# Security Policy

## 🇩🇪 Sicherheitshinweise
Ich nehme Sicherheit und verantwortungsbewussten Umgang mit diesem Projekt ernst.
Bitte melde Sicherheitsprobleme **nicht öffentlich** über Issues oder Pull Requests.

### Wie du Sicherheitsprobleme melden kannst
Wenn du eine Schwachstelle, ein sicherheitsrelevantes Verhalten oder ein Risiko im Zusammenhang mit DNS / frAIme findest, kontaktiere mich direkt:

**[schltdns@gmail.com]**

Bitte gib so viele Informationen wie möglich an, damit ich das Problem nachvollziehen und prüfen kann.

### Reaktionsprozess
1. Ich bestätige den Eingang deiner Meldung so schnell wie möglich.
2. Ich prüfe das Problem und gebe eine Einschätzung.
3. Ich informiere dich über den Status und geplante Maßnahmen.
4. Ich veröffentliche sicherheitsrelevante Fixes erst, wenn das Risiko behoben ist.

Danke, dass du dazu beiträgst, dieses Projekt sicher zu halten.
